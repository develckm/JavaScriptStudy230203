프로토타입 기반 언어

  • 타입을 명시하면 프로토타입 객체를 생성하는 언어
  • 객체 생성시 프로토타입 객체를 기반으로 생성한다.
  • 타입간의 상속은 프로토타입간의 체인으로 구현한다.(__proto__)
  • 자바스크립트에서는 function 을 선언하면 프로타입객체가 만들어져서 함수형 언어라고도 부른다.
  • 이름 있는 함수 (기명함수) : js 엔진 (V8) 코드를 실행하기 전에 해당 함수를 프로토타입 객체로 생성=>함수의 호이스팅
  • 이름이 없는 함술 (익명함수) : 코드를 실행하면서 익명함수를 프로토타입 객체로 생성 (함수의 호이스팅을 막고 최초실행 속도를 빠르게 하기 위해 )
  • 이름이 있는 함수를 변수에 대입(기명함수표현식) : 익명함수와 동작이 같은데 재귀호출이 가능하다.

자바에서 타입(클래스)을 정의하는 방법

        class Type{
            String name;
            int age;
            Type(String name,int age){
                this.name=name;
                this.age=age;
            }
            Type(String name){
                this.name=name;
                this.age=25;
            }
        }

자바스크립트에서 타입(기명함수)을 정의하는 방법1

        function Type(name,age){
            this.name=name;
            this.age=age;
        }

자바스크립트에서 타입(익명함수)을 정의하는 방법2

호이스팅현상이 없고 js 최초실행이 빠르다.

        const Type=function (name,age){
            this.name=name;
            this.age=age;
            this.hello=function(){
                new Type(); (X) 불가능
            }
        }

자바스크립트에서 타입(기명함수 표현식)을 정의하는 방법3

호이스팅현상이 없고 js 최초실행이 빠르고 재귀호출이 가능

        const Type=function name(name,age){
            this.name=name;
            this.age=age;
            this.hello=function(){
                new name(); (O) 가능
            }
        }

자바스크립트에서 타입(class)을 정의하는 방법3

호이스팅 현상이 없고, 객체지향 문법을 사용해서 상속할 수 있다.

타입의 멤버(프로퍼티,속성,필드)를 정의할때 전역변수에서 this 를 사용할 수 없다. (함수나 생성자에서 사용가능)

            job="학생";

            }

            func=()=>{}
